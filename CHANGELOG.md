# 更新日志

## [0.2.6] - 2026-05-24

### 新增
- ⌨️ 新增 `DeepSeek: 触发代码补全` 命令，支持在键盘快捷键设置中自由绑定触发键
- ⚙️ 设置页新增 `triggerKey` 配置项，提供快捷键修改指引入口

## [0.2.5] - 2026-05-24

### 优化
- ✍️ 生成提交信息时实时流式写入 SCM 输入框，用户可看到逐字出现的效果

## [0.2.4] - 2026-05-24

### 修复
- 🐛 修复防抖期间 CancellationToken 过期导致补全结果被丢弃的问题
- 🔔 API 请求失败时添加错误提示，方便排查网络/密钥问题

### 新增
- 🪵 触发补全时状态栏显示 "DeepSeek 补全已触发" 提示（停留 2 秒），方便验证是否正常触发

## [0.2.3] - 2026-05-23

### 变更
- 🔀 调整设置项顺序，优化配置体验

## [0.2.2] - 2026-05-23

### 变更
- 👁️ 设置页新增 API 密钥状态指示器，实时显示"已配置/未配置"

## [0.2.1] - 2026-05-23

### 变更
- ⚙️ 设置页新增 API 密钥管理区域，支持一键设置/清除

## [0.2.0] - 2026-05-23

### 变更
- 🔐 API 密钥改用 SecretStorage 安全存储（OS 凭据管理器）

- ➕ 新增 `DeepSeek: 设置 API 密钥` / `清除 API 密钥` 命令

## [0.1.5] - 2026-05-23
